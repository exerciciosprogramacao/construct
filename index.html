📘 Documentação do Projeto - Dino

🎮 Jogo

👉 Esse jogo foi feito com o Tamanho de exibição de tela de 854 x 480.

👉 Comece inserindo todos os personagens/objetos e o cenário no jogo.

👉 Todos os objetos deste jogo são do tipo Sprite menos o teclado e tempo.

👉 O dino deve ter uma animação de correndo e outra de abaixar.

👉 O cactus deve ter uma animação de cactos e outra de passaro.

👉 Se você usou os sprites fornecidos, todas as colisões "hitbox" já devem estar corretas, assim como os pontos de imagem.

👉 Lembre-se de ajustar as elevações Z para que tudo fique certinho na tela.

👮‍♀️👮‍♂️ Verifique tudo certinho antes de continuar!.

🦖 Pular e abaixar

1 - Teclado: Ao pressionar Barra de espaço      | dino: Simular Plataforma pressionando Pulo

👉 Na linha 1 temos que simular o comando de pulo do comportamento plataforma.

👉 Linha 2 e 3 trocamos a animação que temos dentro do personagem dino.

⚙ Gerador

Gerador não precisa de nenhum comportamento.

⚙ Gerar o cactus

💾 Vamos começar criando uma variável global chamada tipo.

🌎 Global: número tipo = 0

4 - Sistema: A cada 3 segundos                  | Sistema:"definir valor" Definir tipo para escolher(0,1,2,3,4,5) "escolher é choose"
                                                | gerador: "dar origem" Gerar cactus na camada 0 (ponto X=0, hierarquia: Falso)

👉 Temos 5 cactus dentro da nossa animação, equivalentes do 0 ao 4.

👉 O número 5 é nosso passaro, por isso colocamos de 0,1,2,3,4,5.

👉 O escolher "choose". Vai escolher esses numeros de 0 a 5.

👉 Também temos a geração dos cactus, "dar origem a outro objeto".

👉 Conforme o choose escolhe um numero ele defini o frame da animação do cactus para esse número, podendo mudar a cada 3 segundos.

🎋 Cactus


👉 Comportamentos 

Comportamento: MovePara

Velocidade máxima: 650
Aceleração: 0
Desaceleração: 0
Velocidade de rotação: 0
Definir ângulo: Ativado (✔)
Parar em sólidos: Ativado (□)
Habilitado: Ativado (✔)

👉 Mover para serve para pordermos programar uma movimentação ou a chegada até um local.

🎋 Mover e trocar cactus

5 - Sistema: "comparar variável" tipo = 5       | cactus: Definir animação para "passaro" (executar do início)
                                                | cactus: Definir Invertido
                                                | cactus: Definir Espelhado

6 - cactus: Ao criar                            | cactus: Definir quadro da animação para tipo "variável"  
                                                | cactus: MoverPara posição (-50, Self.Y) (Direto) "-50 no x" "Self.Y no Y" 
                                                | cactus: Definir Invertido  
                                                | cactus: Parar animação 

7 - cactus:"Mover para" Ao chegar               | cactus: Destruir
    cactus: "comparar x" X <= -50

👉 Se der o numero 5 na variável tipo, definimos a animação para "passaro".

👉 Na linha 6 é onde realmente acontece a mudança do frame da animação, mudando os cactus.

👉 Essa linha também move o cactus. Estamos mandando ele mover até a posição no fim da tela, atrás do dino.

👉 Na linha 7 verificamos se ele chegou na posição x, se chegou destrói.
